# Intelligent Fusion of Heterogeneous Data for Terrorist Organization Identification

Code and data for the paper: **"Intelligent Fusion of Heterogeneous Data for Terrorist Organization Identification and Attack Attribution"**

## Structure
- `Data_preprocessing_Pipline/` — GTD data preprocessing pipeline (distribution-aware imputation, reverse geocoding, feature reduction)
- `NER_Attribution_Models/` — NER model (RoBERTa-BiLSTM-CRF-Gazetteer), multi-scale classification, and LLM comparison

## Dataset
The Global Terrorism Database (GTD) is available from [START](https://www.start.umd.edu/gtd/)

## Requirements
See `requirements.txt` in each folder.
